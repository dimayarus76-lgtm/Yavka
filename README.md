# Явка (Yavka)

Приложение для организации встреч и сбора компаний в **Ярославле**.

**Анонимная геосоциальная сеть** с картой, чатами, голосовыми комнатами и оффлайн-сборами.

## Основные возможности
- Анонимная авторизация (ник + аватар)
- Интерактивная карта Ярославля (Яндекс.Карты)
- Создание и присоединение к "сборам"
- Приватные и групповые чаты
- Голосовой ведущий
- Игры, квесты, донаты
- Реклама локальных заведений

## Технологии
- **Frontend**: React + React Native (Expo)
- **Backend**: Node.js + NestJS
- **БД**: PostgreSQL + PostGIS (Yandex Cloud)
- **Maps**: Yandex Maps API
- **Realtime**: Socket.io
